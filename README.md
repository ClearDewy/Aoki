## 二、部署
1. 安装docker
2. 在`aoki/aoki-deploy/standAlone`下执行`docker-compose up -d`
3. 在浏览器访问`localhost`

## 三、更新

请在docker-compose.yml当前文件夹下执行`docker-compose pull`拉取最新镜像，然后重新`docker-compose up -d`即可。
